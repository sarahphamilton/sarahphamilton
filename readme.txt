QUICK EDITS

4. Upload all three files together; do not rename them unless you also update the links.

RECOMMENDED HOSTING
Netlify is the simplest nontechnical option:
1. Create a free Netlify account.
2. Create a new site by manual deployment.
3. Upload this entire folder or the ZIP contents.
4. Under Domain management, add a domain you own.
5. Follow the DNS instructions Netlify provides.
